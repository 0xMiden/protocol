# Miden Documentation (Dev Preview)

This is a local Docusaurus preview of the documentation that lives in this repository. In production these docs are aggregated into the [`miden-docs`](https://github.com/0xMiden/miden-docs) repository, where the protocol and standards sections appear alongside docs from other vendor repos.

## Sections

- [Protocol](protocol/index.md): the Miden protocol's core concepts (accounts, notes, assets, transactions, state, blockchain).
- [Standards](standards/index.md): reusable account components published by the `miden-standards` crate.
